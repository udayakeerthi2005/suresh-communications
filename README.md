# Suresh Communications Website

## 📌 Overview

This is a business website developed for **Suresh Communications**, a stamp vendor and document service provider in Hyderabad.

The website provides essential information about services, contact details, and location for customers.

---

## 🚀 Live Website

🔗 https://udayakeerthi2005.github.io/suresh-communications/

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Google Maps Embed

---

## ✨ Features

* Responsive design
* Full-width Google Maps integration
* Click-to-call phone feature
* Clean and simple UI for business use
* Fast loading static website
